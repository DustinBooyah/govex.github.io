Affordability Index for Mid-Sized Cities

Affordable housing is a top priority for many cities, so the Center for Government Excellence (GovEx) created this data visualization so users can explore housing affordability trends in select mid-sized U.S. cities. It includes trends in homeownership, rental markets, demographic shifts, employment and income. It also includes an “All Cities” tab for cities to compare themselves to one another. The Affordability Index is not exhaustive but will be updated with more cities involved in the What Works Cities initiative. Data Source: American Community Survey 5-Year Data Profiles
